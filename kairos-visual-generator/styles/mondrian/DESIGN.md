# Mondrian / De Stijl Design System

## 风格定位

蒙德里安风格 + 红黄蓝白黑色块分割 + 黑色结构线 + 非对称网格 + 主题文字重构 + 文字隐喻表达 + 高级现代主义海报感。

适合：设计、艺术、建筑、构成、创意、视觉、品牌、美学。

## 视觉 DNA

- **色彩哲学**：严格五原色（红、黄、蓝、白、黑），不允许其他颜色。可少量灰色/米灰作为缓冲。
- **结构元素**：黑色粗线和细线构成的秩序分割、非对称几何网格、平涂色块、明确边界。
- **质感**：高级印刷海报感、平面化、干净、克制。
- **信息密度**：极低，标题即全部信息。标题是画面核心视觉资产。

## 色彩系统

严格五原色系统：

| 语义角色 | 说明 | 允许范围 |
|----------|------|----------|
| red | 蒙德里安红 | #DE3831 附近 |
| yellow | 蒙德里安黄 | #F7D44C 附近 |
| blue | 蒙德里安蓝 | #2F5DA6 附近 |
| black | 结构线和文字色 | #1A1A1A |
| white | 背景色 | #FAFAF5 附近 |
| gray | 缓冲色（可选） | 少量使用 |

**禁止**：渐变、霓虹色、金属质感、玻璃拟态、复杂阴影、赛博朋克光效、水彩、手账质感。

## 字体配方

使用几何无衬线或 Bauhaus 风格字体：

```css
--font-display-zh: "Noto Sans SC", sans-serif;
--font-display-en: "Bauhaus 93", "Futura", sans-serif;
--font-text-zh: "Noto Sans SC", sans-serif;
--font-en: "Inter", "Helvetica Neue", sans-serif;
```

规则：
- 中文使用现代黑体、粗黑体、几何黑体
- 英文可以使用高对比 Serif、几何无衬线、Bauhaus 风格
- display weight 可以较重（700），但必须保持可读性
- 标题必须足够大，成为第一视觉焦点

## 网格系统

- 非对称几何网格（非 12 列标准网格）
- 黑色粗线（3-6px）和细线（1-2px）构成秩序分割
- 色块参与标题设计，不只做背景
- 网格比例遵循 De Stijl 经典比例（黄金比例、1:1:2 等）

## 文字重构

Mondrian 的文字重构是最高优先级：

- 标题必须成为蒙德里安式色块分割系统的一部分
- 文字笔画与黑色结构线连接
- 文字内部空间被红黄蓝白黑色块重新分割
- 部分笔画被几何块面替代
- 文字边界与矩形网格互锁
- 标题跨越多个网格单元

文字必须同时具备：
- 清晰可读
- 结构感强
- 色块重构明显
- 内部包含主题隐喻
- 与黑色网格线互相咬合
- 具有现代主义秩序感

## 质感规则

**必须**：
- 平涂色块
- 黑线清晰有力量
- 大面积留白
- 色块边界清楚
- 构图准确

**禁止**：
- 强渐变
- 霓虹色
- 金属质感
- 玻璃拟态
- 复杂阴影
- 油腻光效
- 赛博朋克光效
- 水彩
- 手账质感
- 儿童感

## 反模式

- 随机色块拼贴
- 儿童积木感
- 普通 PPT 模板
- 复杂信息图
- 杂志拼贴
- 网页 UI
- 赛博朋克
- 插画风
- 小红书模板风
- 卡通风
- 电商详情页风
- 电影海报风
- 真实摄影风
- 3D 渲染风

## 专项 QA 规则

- 色块比例是否符合 De Stijl 经典比例
- 文字是否融入色块网格
- 是否保持五原色系统（无其他颜色）
- 黑色结构线是否清晰有力
- 标题是否是第一视觉焦点
- 整体是否像高级现代主义海报
- 文字是否清晰可读
