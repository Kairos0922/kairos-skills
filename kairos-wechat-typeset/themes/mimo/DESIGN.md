# Xiaomi MiMo DESIGN.md

## Theme Position

Xiaomi MiMo 用于 AI、模型发布、技术报告、产品研究和数据型文章。它不是复刻 MiMo 页面，而是抽取“清晰、理性、技术发布、模型报告”的气质，转译为微信公众号移动阅读版式。

## Visual Language

- 关键词：技术白、数据层级、蓝色焦点、理性、清晰、报告感。
- 适合：AI 模型解读、产品发布、benchmark、工程方法论、研究摘要。
- 禁止：过度拟物、复杂科技渐变、霓虹发光、营销海报式 hero。

## Tokens

- Paper: very light cool gray.
- Ink: near-black blue gray.
- Accent: single restrained blue, used sparingly for links, highlights, and tip borders.
- Font: system sans for high information clarity; mono for code/data labels.

## Components

- H1: product/research report title, dense but breathable.
- H2 numeric: compact data-report section marker.
- Paragraph: slightly tighter than Song theme, keeps long technical prose readable.
- Quote/callout: white card-like paper surface with thin rule; still inline-only.
- Code/table: higher contrast than Song, suitable for commands and metrics.

## Mobile Rules

- Prioritize scanability for technical readers.
- Long English tokens must remain readable and not break layout.
- Tables degrade into stacked cards, one row per card.
- Accent color may not dominate the article.

## Quality Gates

- All styles inline.
- No external CSS or user-defined tokens.
- Must pass representative rendering, verification, golden comparison, and mobile review before registry exposure.
